# Apresentação da solução Nulltriverso (IMC)

Resumo e roteiro para demonstrar o primeiro módulo móvel do Nulltriverso: uma calculadora de IMC offline, com gauge e linha ilustrativa, construída em React Native com Expo.

## Roteiro sugerido de apresentação
- Contexto: ecossistema Nulltriverso (tecnologia + nutrição) e objetivo da prova de conceito de IMC.
- Flow principal: preencher nome, peso (kg) e altura (cm) → validação imediata → cálculo e classificação conforme OMS.
- Persistência: reabrir o app mostrando o último cálculo salvo no `AsyncStorage`.
- Visualizações: gauge semicircular e linha de tendência mockada para discutir evolução.
- Usabilidade: teclado numérico, mensagens claras e layout responsivo.

## Como gravar a demonstração
1. `cd Nulltriverso/frontend && npm start`
2. Escanear o QR Code com o Expo Go (Android/iOS) ou abrir no emulador.
3. Seguir o roteiro acima, destacando validações e persistência.
